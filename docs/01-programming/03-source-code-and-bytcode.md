# Source Code and Bytecode

---

## **Source Code**

**Definition:**
Source code is the **human-readable instructions** that a programmer writes using a programming language (like Python, C, or Java). It’s the “recipe” for a program, written so humans can read, understand, and modify it.

**Key Points:**

- Written in a **high-level programming language**.
- Must be **translated** (compiled or interpreted) into machine code for execution by a computer.
- Typically stored in **text files** with extensions like `.py`, `.c`, `.java`, etc.
- Contains **logic, control structures, functions, and declarations** that define the program’s behavior.

**Example (Python Source Code):**

```python
name = input("Enter your name: ")
print(f"Hello, {name}!")
````

- Human-readable and editable by programmers.
- Computers **cannot execute it directly**; it requires an interpreter or compiler.

---

## **Bytecode**

**Definition:**
Bytecode is an **intermediate, lower-level representation** of source code. It is **not human-readable** but also **not tied to specific hardware**. Bytecode is designed to be executed by a **virtual machine (VM)**, such as the Java Virtual Machine (JVM) or Python’s PVM.

**Key Points:**

- Generated by **compiling source code** into an intermediate format.
- Portable across platforms because the VM interprets or JIT-compiles it to machine code.
- Faster than interpreting source code line by line, but slightly slower than native machine code.
- Enables languages like Java, Kotlin, or C# to run on **any system with a compatible VM**.

**Example (Java Bytecode Flow):**

1. Write Java **source code** (`Hello.java`):

```java id="cv4fg4"
class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, world!");
    }
}
```

1. Compile to **bytecode** (`Hello.class`):

```bash id="dw1x0c"
javac Hello.java
```

- The `.class` file contains bytecode that **the JVM executes** on any system.

## **Key Takeaway**

- **Source code** → what you write as a programmer, human-readable and editable.
- **Bytecode** → the portable intermediate form executed by a virtual machine, bridging the gap between source code and machine code efficiently.
