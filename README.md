## 🚀 Overview

**Pythonista** is a structured learning platform designed to teach Python through **progressive modules and real projects**.
Instead of isolated tutorials, the curriculum focuses on **building real applications while learning core programming concepts**.

By completing this journey you will gain experience in:

* Python fundamentals
* automation and scripting
* API development
* data processing
* testing and debugging
* software architecture

## ⚡ Quick Start

Follow these steps to run the documentation locally.

### Requirements

| Tool    | Version | Purpose                |
| ------- | ------- | ---------------------- |
| Python  | 3.12+   | Language runtime       |
| Git     | Latest  | Version control        |
| Node.js | 18+     | Documentation platform |

Downloads:

* [https://www.python.org/downloads/](https://www.python.org/downloads/)
* [https://git-scm.com/](https://git-scm.com/)
* [https://nodejs.org/](https://nodejs.org/)

---

### Clone the Repository

```bash
git clone https://github.com/hetfs/Pythonista.git
cd Pythonista
```

---

### Install Dependencies

```bash
npm install
```

---

### Start the Documentation Server

```bash
npm start
```

Open:

```
http://localhost:3000
```

The site automatically reloads when documentation files change.

## 🎯 Learning Path

The curriculum progresses through key stages of Python development.

```
Fundamentals
   ↓
Control Flow
   ↓
Functions
   ↓
Data Structures
   ↓
File Handling
   ↓
Object-Oriented Programming
   ↓
Modules & Packaging
   ↓
Intermediate Python
   ↓
Working with APIs

Specialization Tracks
```

## 📈 Recommended Learning Roadmap

| Week | Focus          | Goal                                        |
| ---- | -------------- | ------------------------------------------- |
| 1–2  | Fundamentals   | Learn syntax and basic programming concepts |
| 3–4  | Intermediate   | Build small practice projects               |
| 5–8  | Specialization | Choose Web, Data, or Automation             |
